# Как использовать навык Business Analyst Toolkit

## Быстрый старт

Эй, Клод — я только что добавил навык «business-analyst-toolkit». Можешь проанализировать этот набор данных?

## Примеры вызовов

### Пример 1: Базовое использование
```
Эй, Клод — я только что добавил навык «business-analyst-toolkit». Можешь проанализировать этот набор данных?
```

### Пример 2: Конкретный рабочий процесс
```
Эй, Клод — я только что добавил навык «business-analyst-toolkit». Можешь помочь мне с [название рабочего процесса]?
```

### Пример 3: Интеграция с другими навыками
```
Эй, Клод — я только что добавил навык «business-analyst-toolkit». Можешь использовать его вместе с сопутствующими навыками для предоставления полного решения?
```

## Что необходимо предоставить

При использовании этого навыка предоставьте следующее:

- **Набор данных**: Ваши файлы данных (CSV, JSON, Excel) или данные для подключения к базе данных.
- **Контекст** (опционально): Бизнес-цели, ключевые интересующие метрики.
- **Ограничения** (опционально): Требования к производительности, соображения конфиденциальности данных.

## Что вы получите

Этот навык обеспечит:

- **Отчеты об анализе**: Статистические сводки, анализ трендов, инсайты.
- **Визуализации**: Диаграммы и графики, демонстрирующие ключевые выводы.
- **Рекомендации**: Практические шаги, основанные на данных.
- **Автоматизированные инструменты**: 7 Python-скриптов для обработки и анализа данных.

## Доступные Python-инструменты

Этот навык включает в себя следующие Python-инструменты:

- **charter_builder.py**: Charter Builder — генерация уставов проектов по улучшению процессов.
- **gap_analyzer.py**: Gap Analyzer — выявление недостающих элементов в документации процессов.
- **improvement_planner.py**: Improvement Planner — генерация планов улучшения процессов на основе GAP-анализа.
- **kpi_calculator.py**: KPI Calculator — расчет KPI процессов и показателей эффективности.
- **process_parser.py**: Process Parser — универсальный парсер документации процессов.
- **raci_generator.py**: RACI Generator — создание матриц RACI на основе документации процессов.
- **stakeholder_mapper.py**: Stakeholder Mapper — создание карт стейкхолдеров и планов взаимодействия.

Вы можете запускать эти инструменты напрямую:

```bash
python skills/product-team/business-analyst-toolkit/scripts/charter_builder.py --help
```

## Советы для достижения лучших результатов

1. **Будьте конкретны**: Предоставляйте четкие и подробные требования.
2. **Дайте контекст**: Включите релевантную фоновую информацию о вашем проекте.
3. **Итерируйте**: Начните с точечного запроса, затем уточняйте его на основе первых результатов.
4. **Комбинируйте навыки**: Этот навык хорошо работает в сочетании с другими продуктовыми навыками.
