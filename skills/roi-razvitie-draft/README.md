# roi-razvitie-draft

Скилл для Cursor / Claude Code: генерация драфта документа к еженедельной встрече «Roi Развитие» (среда, продукт Roi Navigator).

**Триггеры:** запрос драфта к встрече, к среде, «Roi Развитие», подготовка к еженедельной встрече.

**Выход:** markdown-документ с заголовком `DD.MM Roi Развитие`, блоком «К обсуждению» и блоком «Заметки».

Опционально: файл `past-meetings-sample.md` с текстом прошлых встреч — скилл использует его для структуры и типичных пунктов.
