# Workflow 4: Write Full Article

**Purpose:** Написать полную статью по структуре Julian Shapiro (outline → draft)

**When to Use:**
- Intro протестирован и одобрен (из workflow 3)
- Готов писать развёрнутую статью
- Нужно структурировать большой объём информации

**Prerequisites:**
- State файл с intro и test results
- Понимание objective (что хочешь донести)
- Готовность к iterative drafting

### Step 3: Create Outline (Supporting + Resulting Points)

**Description:** Создай outline по методу Julian Shapiro

**Actions:**

Объясни структуру:
```
По Julian Shapiro, outline строится из:

1. SUPPORTING POINTS — Что нужно чтобы аргумент работал
   "Если читатель не знает/не верит в [X], мой аргумент не сработает"

2. RESULTING POINTS — Что следует из аргумента
   "Если читатель принял мой аргумент, что дальше?"
```

Предложи структуру:
```
На основе твоего objective, вот outline:

━━━━━━━━━━━━━━━━━
📖 OUTLINE

[Intro - уже готов]

SUPPORTING POINTS (Почему это важно):
1. [Point 1] — Объясняет контекст/проблему
2. [Point 2] — Показывает почему current approach не работает
3. [Point 3] — Подводит к твоему решению

CORE ARGUMENT (Твой подход):
4. [Main point] — Твоё решение/метод/инсайт
5. [Evidence] — Proof/example/case study

RESULTING POINTS (Что делать):
6. [Action 1] — Первый шаг
7. [Action 2] — Следующий шаг
8. [Optional: Common mistake] — Чего избегать

CONCLUSION:
9. Poignant takeaway + next steps
━━━━━━━━━━━━━━━━━

Что скажешь? Добавить/убрать/изменить пункты?
```

**Expected Outcome:** Outline одобрен пользователем

### Step 4: Draft Section by Section (with Humanize-Content)

**Description:** Интерактивно напиши каждую секцию outline, используя естественный человеческий стиль

**CRITICAL: Humanize-Content Writing Guidelines**

При написании каждой секции используй эти принципы:

**Core Identity:**
- Пиши как реальный человек объясняет что-то другу
- Используй естественный, разговорный стиль
- Приоритет: clarity + simplicity + relatability

**Writing Principles:**
1. **Short sentences** — Разбивай длинные предложения на короткие
2. **Active voice** — Избегай passive constructions
3. **Everyday language** — Никакого corporate speak или жаргона
4. **Vary sentence structure** — Не повторяй одинаковые конструкции
5. **Show, don't tell** — Конкретные детали вместо generic descriptions
6. **Natural flow** — Как ты говоришь, не как AI пишет

**For each section:**

1. **Покажи что пишем:**
```
━━━━━━━━━━━━━━━━━
Секция: [название секции]
Цель: [что донести в этой секции]
━━━━━━━━━━━━━━━━━
```

2. **Задай вопросы:**
```
Что ты хочешь сказать в этой секции?
- Есть ли пример который иллюстрирует это?
- Есть ли данные/статистика?
- Есть ли история которая делает это запоминающимся?
```

3. **Напиши черновик секции с humanize-content:**

**Apply these rules while drafting:**

✅ **DO:**
- Use varied vocabulary (avoid repeating same high-frequency words)
- Mix short and long sentences for rhythm
- Include specific, vivid details ("The garden was a vibrant tapestry of blooming flowers")
- Use active voice ("The team decided" not "The decision was made by")
- Add occasional rare words when appropriate ("exhilarating" instead of just "exciting")
- Write as if talking to a friend

❌ **AVOID:**
- Repetitive sentence structure ("She went... She bought... She returned...")
- Excessive connectives ("and... and... and...")
- Generic descriptions ("The garden was beautiful")
- Predictable sentence openers (starting every sentence the same way)
- Overly passive voice ("It was decided that...")
- Over-optimization (too polished = not human)
- Filler words ("actually", "very", "basically")
- Predictable clichés ("It was a dark and stormy night")
- Overly simplistic transitions ("then", "next")
- Over-summarization (elaborate with examples!)
- Overuse of anthropomorphism
- Excessive enthusiasm (!!!)
- Vague placeholders ("some people think...")

**Example Transformation:**

❌ AI-style:
"This is a very good and very interesting idea. The event was exciting and fun. He was tired and he wanted to rest and he didn't feel like talking."

✅ Human-style:
"This idea is intriguing and genuinely impressive. The event was exhilarating, a rare blend of thrill and enjoyment. Exhausted, he wanted to rest and preferred silence."

```
Вот черновик:

[3-5 параграфов - написанные в human style]

Что скажешь?
OPTIONS:
- "Отлично, дальше"
- "Переписать" → уточни что
- "Добавить пример"
- "Сделать более human" → применю humanize-content правила сильнее
```

4. **Repeat для всех секций**

**Expected Outcome:** Полный черновик написан секция за секцией в естественном человеческом стиле

### Step 5: Write Conclusion

**Description:** Создай сильное завершение

**Actions:**

Структура conclusion:
```
Conclusion состоит из:

1. POIGNANT TAKEAWAY — Главная мысль в 1-2 предложениях
   "Не просто summary, а insight который останется с читателем"

2. NEXT STEPS — Что делать прямо сейчас
   "Конкретное actionable действие"
```

Предложи conclusion:
```
━━━━━━━━━━━━━━━━━
CONCLUSION

[Poignant takeaway - essence статьи]

[Next steps - что делать]

[Optional: Final thought - оставляет impression]
━━━━━━━━━━━━━━━━━

Это сильное завершение?
```

**Expected Outcome:** Conclusion написан
