# POS WS03 — Claude Code, автоматизация, практический показ

## Video
[![POS WS03 — Claude Code, автоматизация, практический показ](https://img.youtube.com/vi/bzC3l5Ynv2w/maxresdefault.jpg)](https://youtube.com/watch?v=bzC3l5Ynv2w)

## Summary
Третий воркшоп POS-спринта посвящён архитектуре персональной операционной системы через призму **семи слоёв**: оператор, стратегия, долгая память, координация, правила, рабочая среда и исполнение. Сергей Хабаров провёл live-демо настройки **OpenClaw** как Telegram-бота с интеграцией GitHub Issues, показал технику долгой автономной работы агента через файлы **plan.md** и **status.md**, объяснил механизм **компактизации контекста** в Codex и почему он превосходит Claude Code для многочасовых задач.

Практическая часть включала пошаговое создание Telegram-группы с ботом, подключение LLM через OpenRouter (модель Kimi), настройку тредов с изолированным контекстом и **метод супервизора** — когда Codex/Claude Code настраивает OpenClaw, а не OpenClaw настраивает сам себя. Обсуждались скиллы (как их выбирать, создавать под себя, оценивать по результату), инструменты end-to-end тестирования (DevTools MCP, Telethon для Telegram), а также вопросы стоимости API и влияния AI на рынок труда разработчиков.

## Tools
- [[OpenClaw]] — Open-source Telegram-бот с LLM-мозгом, работает через OpenRouter. Используется как project-менеджер с тредами и GitHub-интеграцией.
- [[Codex (OpenAI)]] — Терминальный AI-агент от OpenAI. Превосходит Claude Code в компактизации контекста и автономной работе (8-9 часов без деградации).
- [[OpenRouter]] — Провайдер API различных LLM-моделей. Позволяет выбирать модели по цене и возможностям, удобен для экспериментов.
- [[DevTools MCP]] — MCP-инструмент для управления браузером Chrome. Позволяет агенту самостоятельно тестировать веб-страницы (end-to-end тестирование).
- [[Telethon]] — Библиотека для автоматизации Telegram. Позволяет агенту самостоятельно тестировать ботов и управлять аккаунтом.
- [[GitHub Issues/Projects]] — Слой координации задач. Доска задач синхронизируется с агентами, хранит историю решений.
- [[Raycast]] — Лаунчер для macOS. Используется для истории копирования — ключевая функция продуктивности при работе с токенами и ключами.
- [[BotFather]] — Telegram-бот для создания новых ботов. Генерирует токены для подключения к LLM.
- [[Kimi (Moonshot AI)]] — Китайская LLM-модель, дешёвая и достаточная для задач project-менеджера в OpenClaw.
- [[Perplexity]] — AI-поисковик, используется для быстрых вопросов и решения проблем при настройке.
